# 大耳狗雲朵相簿

這個專案現在使用 **Vite** 作為前端開發架構，重新設計成大耳狗風格的相簿網站。頁面提供柔和雲朵系視覺、可上傳多張圖片的相簿、逐張滾輪瀏覽，以及 8-bit 預覽轉原始照片的查看體驗。

## 網頁特色

1. **大耳狗風格版面**：以淺藍、白色、圓角與雲朵感卡片重構首頁視覺。
2. **圖片上傳功能**：使用者可在頁面上傳多張圖片，立即加入相簿。
3. **逐張滾輪相簿**：相簿區支援滑鼠滾輪一張一張往下切換，也可使用上下按鈕瀏覽。
4. **8-bit 預覽**：相簿卡片預覽會先以像素化風格呈現。
5. **點擊看原圖**：點擊任一預覽卡片後，會以彈窗顯示實際照片。

## 開發方式

```bash
npm install
npm run dev
```

## 建置

```bash
npm run build
```

## 主要檔案

- `index.html`：Vite 入口頁面。
- `src/main.js`：網站互動邏輯、圖片上傳、8-bit 預覽與彈窗查看。
- `src/style.css`：大耳狗風格視覺與相簿排版樣式。
