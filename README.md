# 大耳狗雲朵相簿

這個專案已改為 **Vite** 架構，重新設計成大耳狗風格的雲朵相簿。網站支援圖片上傳、以滾輪一張一張往下瀏覽，以及用 8-bit 風格預覽照片；點擊後可開啟原始相片。

## 特色

1. **Vite 前端架構**：使用 `src/main.js` 作為入口，方便後續擴充與部署。
2. **大耳狗風格介面**：以雲朵、淺藍色與圓角卡片重做整體視覺。
3. **圖片上傳**：可一次選取多張圖片，並儲存在目前瀏覽器的 IndexedDB 中。
4. **一張一張滑動瀏覽**：透過滾輪或方向鍵，以整頁滑動方式閱讀相簿。
5. **8-bit 預覽 + 原圖檢視**：相簿中看到的是像素風預覽，點擊後會開啟實際照片。

## 開發方式

```bash
npm install
npm run dev
```

## 打包

```bash
npm run build
```

## 主要檔案

- `index.html`：Vite HTML 入口。
- `src/main.js`：相簿渲染、圖片上傳、IndexedDB 儲存與預覽邏輯。
- `src/style.css`：大耳狗風格版面與互動樣式。
